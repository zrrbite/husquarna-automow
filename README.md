# Husqvarna Automower

Personal documentation for Husqvarna Automower robotic mowers: parts replacement, error diagnosis, and maintenance notes.

Note: Husqvarna is the correct spelling — the repo folder name `husquarna-automow` is kept as-is for convenience.

## Models

- [Automower 310](./automower-310/) — owned

## Structure

Each model folder contains:

- `parts/` — replacement parts, part numbers, suppliers, notes on fit
- `errors/` — error codes, symptoms, root causes, fixes
- `maintenance/` — routine service (blades, bearings, cleaning, winter storage)
