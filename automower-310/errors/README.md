## Common codes (reference)

These are general Automower codes — confirm against the 310 manual before acting.

- **1** — Outside working area (loop signal lost / wire break)
- **2** — No loop signal (charger off, wire break, fuse)
- **3** — Wrong loop signal (swapped A/B at charger)
- **4** — Loop sensor problem
- **5** — Charging station blocked
- **8** — Wrong PIN
- **10** — Upside down
- **12** — Empty battery
- **13** — No drive (stuck, wheel blocked, wheel motor fault)
