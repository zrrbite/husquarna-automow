# Maintenance — Automower 310

Routine service procedures and notes.

## Intervals (rough)

| Task | Interval |
|------|----------|
| Clean chassis + cutting disc | Weekly |
| Inspect blades for chafe/wear | Every 2–4 weeks |
| Replace blades + screws (set) | Every 1–2 months in season |
| Check wheels for debris / free rotation | Monthly |
| Clean charging contacts | Monthly |
| Check cutting-disc bearing for play/noise | Seasonally |
| Winter storage (fully charge, dry, indoors) | End of season |
| Spring wake-up (charge, blade change, firmware check) | Start of season |

## Diagnostic symptoms → where to look

- Uneven cut / mower scalping → blade wear, bent disc, wheel pressure
- Grinding / rattling from underneath → blade screws loose, bearing play, debris
- Short runtimes → battery age, cold weather, heavy grass
- Won't dock → dirty contacts, station misaligned, wire signal
